IMAX GT (15/70)

Not every premium format is a ratio. Disney's Infinity Vision certification (2026) sets a minimum 50ft screen width, requires laser projection, and 7.1-channel-plus immersive audio — but unlike IMAX, it has no fixed aspect ratio requirement at all. It's a performance target, not a frame shape, which is why it doesn't get a tab above.
